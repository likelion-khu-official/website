# 🗺️ 사이트 구조 (IA)

> 🧭 **이 논의는 '수렴'하는 자리예요**
> 새 페이지·메뉴를 벌리기보다, 아래 구조가 맞는지 함께 좁혀가요. 의견 주기 전에 편하게 짚어봐 주세요:
> - 이건 **이 구조에 대한 의견**일까요, 아니면 **새 페이지를 더하자는 제안**일까요?
> - 그 페이지, **지금 없으면 안 될까요, 나중에 더해도 될까요?**
> - 우리 규모(**부원 ~20명 · 글 월 2건**)에서 **정말 필요한** 화면일까요?

## 👥 권한 — 위에서 아래로만 위임돼요
```
👑 최고관리자 (딱 1명)
     │  관리자 임명 · 회수
     ▼
🛡️ 관리자
     │  멤버 등록 · 계정 발급
     ▼
🧑 멤버 ───(로그인)───▶ 멤버면

👤 방문자 ─(로그인 없이)─▶ 공개면
```

## 🌐 공개면  `/`  — 누구나
```
랜딩  /
├─ 히어로 · 소개 · 통계 · 세션 · 연간계획 · 푸터
├─ 프로젝트 요약 ──더보기──▶ 프로젝트 /projects ──▶ 상세 /projects/:id
├─ 운영진 소개
├─ 블로그 요약  ──더보기──▶ 블로그 /blog ──▶ 글 상세+댓글 /blog/:slug
└─ 모집 ┬─ 모집중 ─▶ 지원폼 /apply  (＋개인정보 동의)
        └─ 평소   ─▶ 모집알림 신청
멤버 탭  /members
```

## 🔑 로그인
```
/login  (학번 + 비밀번호)
├─ 첫 로그인 ─▶ 비밀번호 변경 ─▶ 각자 면으로
└─ 비번 분실 ─▶ 관리자가 초기화
```

## 🔒 멤버면  `/member`  — 로그인한 부원
```
├─ 글쓰기 (바로 게시)
├─ 내 글 수정 · 삭제
├─ 내 프로필 편집 (사진 · 입부계기)
└─ 내 프로젝트 등록 · 수정 · 삭제
```

## 🛡️ 어드민면  `/admin`  — 관리자 · 운영 실무
```
├─ 멤버 등록 · 계정발급 · 비번초기화
├─ 운영진 소개 관리 (등록·수정·삭제·정렬)
├─ 블로그 · 댓글 모니터링 (숨김)
├─ 프로젝트 숨김
├─ 지원자 명단 열람
└─ 모집알림 구독자 열람
```

## 👑 최고관리자면  — 최고관리자 · 권한 관리
```
├─ 관리자 임명 · 회수
└─ ＋ 어드민면 운영 실무 전부
```
