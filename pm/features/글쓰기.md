# 글쓰기  🟡 진행

<sup>[⌂ 사이트](README.md) › [멤버 영역](멤버-영역.md)</sup>

> 로그인 부원이 피드 글을 작성(바로 게시).

## 개요
- 로그인한 부원이 블로그 글을 작성한다.
- 제목·본문(이미지 삽입)·썸네일을 쓴다.
- 임시저장·미리보기 지원, 제출하면 승인 없이 바로 게시된다.
- 작성자는 로그인 본인이라 이름·파트가 자동으로 붙는다(사칭 불가).
- 관련 기능: [블로그](블로그.md).

## 진행 상태
매직링크 방식에서 로그인 세션 기반 자유작성으로 전환 중이다. 백엔드는 #115(담당 안시현), 프론트는 #116(담당 박일하)로 진행 중이며 목표는 7/21이다. (이전 #94·#95는 "멤버 로그인" 정책 확정을 기다리며 홀드됐다가 닫히고 이 두 이슈로 재발주됐다.)
