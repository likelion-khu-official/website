# 멤버 관리  🟡

<sup>[⌂ 사이트](README.md) › [어드민](어드민.md)</sup>

> 관리자가 멤버를 등록하고 계정을 발급·수정·오프보딩하는 화면.

## 개요
- 새 부원 등록 → 계정 발급(아이디=학번, 초기 비번=전화번호, 이모지 자동 배정).
- 정보 수정(옵셔널) — 이름·역할 정정, 부적절 시 사진·입부계기 손봄.
- 비밀번호 초기화(분실 시).
- 오프보딩 — 소프트 딜리트(로그인만 막고 기록 유지).
- 관련 기능: [계정 · 인증](계정-인증.md)

## 진행 상태
등록(POST)·수정(PATCH)·비밀번호 초기화 API는 이미 dev에 머지돼 있다(#117, PR #121·#132). 남은 건 오프보딩(소프트 딜리트) 백엔드와 어드민 화면(FE)이다. 이 둘을 #145(담당 장찬욱, ~7/30)로 발주했다 — 학번 로그인은 됐지만 계정을 발급할 이 화면이 없어 아직 아무도 로그인 못 하는 상태라, 이 미션이 로그인을 실사용 가능하게 만든다.
