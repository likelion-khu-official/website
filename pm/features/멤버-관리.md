# 멤버 관리  🟡

<sup>[⌂ 사이트](README.md) › [어드민](어드민.md)</sup>

> 관리자가 멤버를 등록하고 계정을 발급·수정·오프보딩하는 화면.

## 개요
- 새 부원 등록 → 계정 발급(아이디=학번, 초기 비번=전화번호, 이모지 자동 배정).
- 정보 수정(옵셔널) — 이름·역할 정정, 부적절 시 사진·입부계기 손봄.
- 비밀번호 초기화(분실 시).
- 오프보딩 — 소프트 딜리트(로그인만 막고 기록 유지).
- 관련 기능: [계정 · 인증](계정-인증.md)

## 진행 상태
등록(POST)·수정(PATCH) API는 SUPER_ADMIN 전용으로 dev에 배포됐고, 비밀번호 초기화는 PR #121(OPEN)에 있다. 오프보딩(소프트 딜리트)과 어드민 화면(FE)은 미구현. (이전 서술 "멤버 계정 관리 흐름 미구현 🔵"은 2026-07-18 GWT 코드 검증에서 낡은 것으로 확인돼 정정 — 근거는 아래 검증 문서의 "트리 정정" 섹션.)
