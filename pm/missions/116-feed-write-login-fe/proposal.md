## [프론트] 글쓰기 화면 — 링크 없이 바로 쓴다

로그인한 부원이 **버튼만 누르면 바로** 글을 쓰는 화면을 만듭니다. 예전 방식(매직링크로 들어와 작성자를 직접 고르던 것)을 걷어내고, 로그인 상태에서 곧장 작성하며 작성자는 화면에 **자동으로** 표시됩니다.

> **지금 상태** — 현재 화면은 매직링크로 진입해 작성자를 직접 고르는 방식입니다. 앞선 시도 #95는 정책 확정을 기다리며 홀드됐고, 이제 위키로 확정돼 재발주합니다. 백엔드 미션(#115)과 짝입니다.

---

**Target** — 이 미션이 전진시키는 기능 트리 노드: [`글쓰기` 노드 문서](https://github.com/likelion-khu-official/website/blob/dev/pm/features/글쓰기.md) (→ [블로그](https://github.com/likelion-khu-official/website/blob/dev/pm/features/블로그.md)). 정식 스펙은 [서비스 위키](https://github.com/likelion-khu-official/website/wiki). **여기서 무엇·왜의 상세를 본다.**

**Deliverable** — `/member` 글쓰기 화면. 제목·본문·이미지·썸네일 작성 + 임시저장·미리보기. 작성자는 로그인 세션 기준으로 표시. **작성자를 직접 고르는 드롭다운 UI 제거.** 리뷰용 PR(→ dev).

> [!IMPORTANT]
> **Why** — 부원이 번거로우면 글을 안 씁니다. 로그인만 돼 있으면 링크 없이 바로 쓰게 하고, 작성자는 세션에서 자동으로 붙이는 게 확정된 방향입니다(위키).

**Done**

- [ ] 작성자를 **직접 고르는 UI가 없다**(세션 기준으로 자동 표시)
- [ ] 제목·본문·이미지·썸네일·임시저장·미리보기가 동작한다
