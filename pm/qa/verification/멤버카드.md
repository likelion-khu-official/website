### CARD-001 — pass

멤버 카드의 셀프 편집(사진·입부계기 본인 수정) 경로가 스펙엔 있지만, `find backend/src/main/java -iname "*member*"`로 나온 파일을 전부 확인한 결과 코드엔 없다. 유일한 쓰기 경로는 `MemberController.java:28-46`의 `POST /api/admin/members`·`PATCH /api/admin/members/{id}`뿐이고, 둘 다 `@PreAuthorize("hasRole('SUPER_ADMIN')")`가 붙어 있다(28행·38행). `SecurityConfig.java`에도 이 경로를 열어주는 permitAll이 없으므로 익명은 인증 단계에서부터 401 UNAUTHENTICATED로 막힌다 — 카드 정보를 익명은 물론, 멤버 본인조차(셀프 편집 API 자체가 없으므로) 못 고친다.
