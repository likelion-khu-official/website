# 첫 로그인 비밀번호 변경  🟡 진행

<sup>[⌂ 사이트](README.md) › [인증 · 계정](계정-인증.md)</sup>

> 발급받은 초기 비번(전화번호)을 첫 로그인 때 바꾼다.

## 개요
- 첫 로그인 시 전화번호로 된 초기 비밀번호를 새 비밀번호로 한 번 바꾼다 — 전화번호를 그대로 두지 않게.
- 이후엔 바뀐 비밀번호로 로그인한다.

## 진행 상태
어드민 초대 흐름엔 가입 시 비번 설정이 있고, 멤버용 첫 로그인 강제 비번 변경도 #117(PR #121·#132)로 머지됐다 — 서버 필터(`MemberPasswordGuardFilter`)가 첫 로그인 상태에선 비번 변경 외 요청을 막는다.
