# 멋쟁이사자처럼 경희대 — 공식 사이트

멋사 경희대의 공식 웹사이트를 만듭니다.
한 기수가 쓰고 버리는 사이트가 아니라, 동아리가 계속 쌓아 올릴 **자산**을 만듭니다.

---

## 우리가 보는 것

동아리의 활동은 매 기수 반복되지만, 결과물은 흩어지고 사라집니다.
인스타에, 노션에, 개인 폴더에. 검색도 안 되고, 다음 기수로 이어지지도 않습니다.

그래서 세 가지를 목표로 합니다.

1. **멋사의 얼굴** — 누가 와도 5초 만에 "이런 곳이구나"가 전해진다.
2. **휘발되지 않는 방문** — 지금의 관심이 다음 모집으로 이어진다.
3. **쌓이는 구조** — 프로젝트, 사람, 글이 기수를 넘어 누적된다.

---

## 일하는 방식 — AI-native

우리는 개발의 다음 표준을 지금 씁니다.

**사람이 결정하고, AI가 구현합니다.**
PM이 무엇을·왜를 정의하면, 팀원이 어떻게를 정하고 AI와 함께 만듭니다.
사람의 시간은 판단에, 반복은 기계에 씁니다.
