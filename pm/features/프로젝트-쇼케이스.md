## 개요
- 담는 것: 제목 · 한 줄 소개 · 이미지 여러 장(그중 대표 1장 — 목록 카드엔 대표만, 상세엔 전부) · 참여 멤버(이름·파트) · 기수 · 개발 기간 · 기술 스택(태그 여러 개) · GitHub 링크. (상세 설명·발표자료는 처음엔 생략 가능.)
- 방문자는 목록·상세를 열람. 멤버가 본인 참여 프로젝트를 직접 등록·수정·삭제. 관리자는 문제 시 숨김.
- 노출: 랜딩에 대표 카드 몇 개 + 더보기, 전체는 `/projects`.

## 현재 진행 상태
아직 구현이 시작되지 않았다. 전용 서브페이지 디자인 시안 작업(#65)만 열려 있는데, 이마저도 블로그 서브페이지와 묶인 디자인 미션이라 프로젝트 목록·상세 시안이 나오는 단계다(목표 7/8이었으나 아직 OPEN). 백엔드 데이터 API도, 프론트 화면 구현도 착수 전이며 "프로젝트/project/쇼케이스" 키워드로 검색해도 구현 이슈·PR은 없다. 요컨대 기능은 정의됐고 디자인이 그려지는 중일 뿐, 만들어진 것은 없다.
