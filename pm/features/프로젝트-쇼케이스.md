## 개요
- 담는 것: 제목 · 한 줄 소개 · 이미지 여러 장(그중 대표 1장 — 목록 카드엔 대표만, 상세엔 전부) · 참여 멤버(이름·파트) · 기수 · 개발 기간 · 기술 스택(태그 여러 개) · GitHub 링크. (상세 설명·발표자료는 처음엔 생략 가능.)
- 방문자는 목록·상세를 열람. 멤버가 본인 참여 프로젝트를 직접 등록·수정·삭제. 관리자는 문제 시 숨김.
- 노출: 랜딩에 대표 카드 몇 개 + 더보기, 전체는 `/projects`.

## 현재 진행 상태
백엔드 구현이 착수됐다 — #119(담당 장찬욱)가 목록·상세 API와 `shared/types/project.ts` 계약을 함께 만드는 중이며, 이 계약이 나오면 프론트 미션이 뒤따를 예정이다(목표 7/22). 백엔드 컨벤션 리스크 때문에 선우·시현의 PR 리뷰가 머지 조건으로 걸려 있다. 디자인은 전용 서브페이지 시안 작업(#65)이 블로그 서브페이지와 묶여 진행 중이다(목표 7/8이었으나 아직 OPEN). 요컨대 백엔드가 막 시작됐고, 디자인은 진행 중이며, 프론트는 계약이 나온 뒤 시작된다.
